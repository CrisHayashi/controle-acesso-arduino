# Sistema de Controle de Acesso com Registro de Eventos

Projeto acadêmico desenvolvido no curso de Análise e Desenvolvimento de Sistemas.

## Objetivo

Implementar sistema embarcado utilizando Arduino Uno com:

- Teclado matricial 4x4
- Validação de senha
- Acionamento de servo motor
- Registro de eventos
- Comunicação com servidor local
- Evolução futura para nuvem

## Estrutura do Projeto

- `/docs` → documentação técnica
- `/src` → código em linguagem C
- `/wokwi` → arquivos de simulação
- `/server` → servidor local para logs

## Tecnologias Utilizadas

- Arduino Uno
- Linguagem C
- Git
- Wokwi
